备份

准备工作
  1. 设备获取root权限
  2. 在开发者模式中开启usb调试：adb shell
  3. 给予shell以root权限：su
开始备份
  1. 在设备sdcard目录中创建用于存储备份的目录：
    • mkdir /sdcard/000_Backup
  2. 读取设备分区信息，屏蔽掉没有必要的userdata和缓存分区：
1

ls -1 /dev/block/bootdevice/by-name | grep -ixvE "userdata|cache" | while IFS= read -r name; do echo "dd if=/dev/block/bootdevice/by-name/$name of=/sdcard/000_Backup/$name.img" >> /sdcard/000_Backup/001_Backup.sh; echo "fastboot flash $name $name.img" >> /sdcard/000_Backup/002_Restore.bat; done
  1. 使用多出来的sh脚本进行备份（dat脚本用于还原）
    • sh /sdcard/000_Backup/001_Backup.sh
  2. 找到persist.img分区并删除，dat脚本中的最好也删除
    • ps：刷入后可能损毁tee导致安全支付永久失效
  3. 创建MD5值的表格用于日后验证备份文件是否损毁
    • cd /sdcard/000_Backup && md5sum * > /sdcard/000_Backup/003_MD5Hsah.txt
一些优化
  • 剔除官方线刷包中已有的分区（也可以防止还原时可能出现的版本冲突）：
    • 此行命令运行较快
1

sed -i -e '/ super.img/s/^/::/g' -e '/ system.img/s/^/::/g' -e '/ system_a.img/s/^/::/g' -e '/ system_b.img/s/^/::/g' -e '/ vendor.img/s/^/::/g' -e '/ vendor_a.img/s/^/::/g' -e '/ vendor_b.img/s/^/::/g' -e '/ mmcblk0.img/s/^/::/g' -e '/ sda.img/s/^/::/g' -e '/ sdb.img/s/^/::/g' -e '/ sdc.img/s/^/::/g' -e '/ sdd.img/s/^/::/g' -e '/ sde.img/s/^/::/g' -e '/ sdf.img/s/^/::/g' -e '/ sdg.img/s/^/::/g' /sdcard/000_Backup/002_Restore.bat
  • 打包（打包过程很长耐心等待）
    • cd /sdcard && tar -zcpvf PartitionBackup.tgz 000_Backup
恢复
  • 启动到fastboot模式，执行002_backup.bat

LineageOS卡刷方法

  • 刷入boot分区：fastboot flash boot boot.img

  • 刷入dtbo分区：fastboot flash dtbo dtbo.img

    • 跨版本升级需要线刷底包。如安卓13要升级到15，需要线刷14的官方包，再卡刷到安卓15。

卡刷

刷机前的格式化
  • 双清，三清以及四清的区别
  • 双清：选择Dalvik Cache、Cache。
    • 双清只清除缓存，不清除系统和用户数据。双清一般用在相同系统版本进行ota升级时。
  • 三清：选择Dalvik Cache、Cache、Data。
    • 三清后刷机失败，是还可以进入系统的。此时的刷机失败指的是twrp再输入检查时会出现错误
  • 四清：选择Dalvik Cache、Cache、Data、System。（一般不选）
    • 四清是最完整的wipe，把系统、缓存、用户数据等全部清除，清除了之后如果刷机失败，会进不去系统。

获取root权限

twrp修补法
  • 下载面具
  • 将面具后缀名apk改为zip
  • 进入twrp刷入更改后的压缩包
boot修补法
  • 下载adb命令行工具
    • 解压，备用
  • 下载Magisk（面具）
    • Magisk中文网
    • 开源地址
    • 将面具复制到手机里，并且安装

  • 解包：

    1. 下载解包工具
    2. 解压完整包和解包工具（查看解压后的完整包是否有boot.img这个文件，如果没有再进行下一步）
    3. 将完整包payload.bin文件的放在解压包的payload_input文件夹里
    4. 将解包后的boot.img复制到手机里

  • 开始刷入：

    1. 打开面具，点击安装，选择选择并修补一个文件，选择手机里的boot.img文件进行修复
    2. 电脑打开手机内部存储设备，打开Download目录，将magisk_patched-24300_82wTg.img文件复制到adb命令行文件夹的根目录
    3. 将手机关机，按住音量下加开机键进入FASTBOOT模式
    4. 点击解压后的adb命令行工具中的打开CMD命令行.bat文件
      • 输入以下命令：fastboot flash boot <修补后Download目录下的文件名>
      • 见到Finished.说明刷入成功
    5. 输入fastboot reboot重启设备
安装LSPosed模块
  1. 打开面具，点击设置，找到magisk栏，开启zygisk(beta)选项
  2. 下载LSPosed模块：面具是24.1版本以上的下载带zygisk的
  3. 点击模块安装
